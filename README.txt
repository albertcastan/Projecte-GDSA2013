
HOW TO USE:

1. Start Matlab
2. Run "import_data.m" to import all the metadata and train the classifier
3. Run "classify_v2_2.m" to classify new images and export the results to a file text
4. Run "Eval.m" to evaluate the classification and see the results (precision, recall, f-score…)
